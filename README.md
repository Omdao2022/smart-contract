Solidity smart contract for KYC.

## deploy

### using Ignition
```sh
npx hardhat ignition deploy ignition/modules/KYC.ts
```

### using script
```sh
npx hardhat run scripts/KYC.ts
```

## compile

```sh
npx hardhat compile
```

## test
```sh
npx hardhat test
```
